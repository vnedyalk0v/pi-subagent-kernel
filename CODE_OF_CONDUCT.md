# Code of Conduct

Be respectful, constructive, and concise.

Not welcome:

- Harassment, threats, hate, or personal attacks.
- Sharing private information without permission.
- Spam, trolling, or repeated off-topic disruption.

If you see a problem, contact the repository owner or open an issue if it is safe to do so. Security-sensitive concerns belong in `SECURITY.md`.

Maintainers may edit or remove abusive content and block repeat abuse.
